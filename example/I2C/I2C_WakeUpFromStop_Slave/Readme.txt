****************************** 例程描述********************************
功能描述：这是一个I2C1通过主机向从机发送设备地址，从而唤醒在stop模式下的从机。
	此功能需要主机例程与从机例程相互配合使用。
	

时钟配置：
	本例程采用HSI48MHz作为系统时钟，用户如果需要更改系统时钟配置，需要在
	I2C_Config函数之前完成，关于系统时钟如何配置，请参考HK32F0301MXXC用户
	手册或者RCC库函数及RCC相关例程。
	
适用芯片：
	HK32F0301MXXC全系列
涉及外设：
	I2C/RCC/GPIO/UART

注意事项：
	1. 推荐使用MDK5.00或IAR8.22.1及其以上版本编译本工程；
	2. 使用之前检测电源VCC、GND等是否正常。

如何使用：
	1. 找两块用此MCU的开发板，一块下载主机例程，另一块下载从机例程。
	2. 用杜邦线通过I2C1_SCL(PC6)和I2C1_SDA(PC5)将两块板子连接起来。
	
	4. 复位主机，从机收到数据后被唤醒。
